# Симуляция OU-дрейфа (§ 6.2 статьи *Нестационарная предсказательная эффективность*)

Численная иллюстрация режима коллапса (Лемма 2, § 5.1) на каноническом OU-классе § 5.2;
дополнение к PSP-суррогату в `../markov_drift/`, который иллюстрирует режим лишь
качественно. При $\sigma = 0{,}1$ ($\sigma^2/\lambda = 10$) условие OU-concentration
**не выполнено**, поэтому гипотеза S8.1 / (S8.2) Supplementary об адиабатической
асимптотике $(K/2)\ln(\lambda t)$ здесь **не тестируется** — этим заняты
`../markov_drift_ou_iinf/` и `../markov_drift_ou_iinf_adiab/`.

## Модель в одном абзаце

- Конечнозначная марковская цепь на $\Omega=\{1,\dots,k\}$ с $k=8$.
- Внедиагональные логиты $\theta_{ij}(t)$ ($i\ne j$) матрицы переходов
  эволюционируют по независимым SDE Орнштейна–Уленбека
  $d\theta_{ij} = -\lambda(\theta_{ij}-\theta_{ij}^*)\,dt + \sigma\,dW_{ij}$,
  с $\theta_{ij}^*=0$ (равномерная стационарная среда),
  $\theta_{ii}=0$ как якорь для softmax-нормировки строк.
- $P(t) = \mathrm{softmax}_{\text{row}}(\theta(t))$ (по всем $k$ столбцам,
  включая заякоренную диагональ); $K = k(k-1) = 56$ свободных параметров.
- Интегрирование Эйлера–Маруямы с $\Delta t = 1$.
- Наблюдатель удерживает $\hat\theta(t)$ через онлайн-восхождение по
  максимуму правдоподобия на пошаговом лог-правдоподобии с затухающим шагом
  $\eta_t = c/\sqrt{t}$; это байесовская аппроксимация с диффузионным prior для
  OU-постериора, упомянутая в § 5.2.
- Предсказательная информация на горизонте $\tau=1$:
  $I_{\mathrm{pred}}(t,\tau) = I_{\mathrm{opt}}(P(t),\tau)
   - \mathbb{E}_{X_t\sim\pi(t)} D_{\mathrm{KL}}(P(t)^\tau \,\|\, \hat P(t)^\tau)$,
  с обрезкой по нулю.
- **Эффективность**: $\eta = I_{\mathrm{pred}}/I_{\mathrm{mem}}$ с *информационным* знаменателем
  $I_{\mathrm{mem}}(t) = (K/2)\ln(\max(N_{\mathrm{obs}},e))$, $N_{\mathrm{obs}}=t$ — MDL-оценщик растущей памяти
  (байесовская сложность $K$-параметрической модели).
- **Каноническая ностальгия** — недобор прогноза $\nu^{\mathrm{op}}(t) = 1 - I_{\mathrm{pred}}/I_{\mathrm{opt}}$
  (различает режимы, ведущая мера); структурный балласт $\nu^{\mathrm{Still}} = 1 - \eta$ — вторичный (структурно $\approx 1$).

## Почему эти значения параметров

- $\lambda = 10^{-3}$, $\sigma = 0.1$: стационарная дисперсия OU
  $\sigma^2/(2\lambda) = 5$; это *сильный режим флуктуаций*
  ($\sigma^2/\lambda = 10$), **а не** режим OU-concentration / адиабатический —
  см. supplementary § S4.2. Мы выбираем эту точку как наиболее благоприятную
  рабочую: она удерживает цепь действительно информативной
  ($I_{\mathrm{opt}} \sim 0.7$ нат/шаг), тогда как адиабатическое
  $\sigma \sim 3\cdot 10^{-3}$ обрушивает $I_{\mathrm{opt}}$ в численный шум.
  Верхняя граница окна переходного процесса $1/\sigma^2 \approx 10^2$, поэтому
  $T = 2\cdot 10^4$ прогоняется далеко за строгой границей транзиента; тем не
  менее в этой точке концентрация постериора остаётся ведущим эффектом.
- $T = 2\cdot 10^4$, $50$ независимых реализаций.
- Окно подгонки $[1000, 15000]$: $t \gg \tau_E = 10^3$ (исключает старт-ап).
- Начальные логиты $\theta(0) \sim \mathcal{N}(0, 0.1^2)$.

## Как запустить

```bash
pip install -r requirements.txt
python main.py
```

Время прогона — примерно 5–15 минут на современном ноутбуке (доминируется
пошаговым softmax + стохастическим сэмплированием 8x8 на
$50 \times 2\cdot 10^4 = 10^6$ шагах цепи).

## Результат

Роббинс–Монро под дрейфом **замерзает**: $\nu^{\mathrm{op}}$ растёт $0{,}567 \to 0{,}892$ (t = 2200 → 20000),
$\langle\nu^{\mathrm{op}}\rangle = 0{,}820$, $\eta \approx 5\cdot10^{-4}$. Постоянный шаг $s=0{,}3$ **отслеживает**:
$\nu^{\mathrm{op}}$ удерживается $\approx 0{,}33$, $\langle\nu^{\mathrm{op}}\rangle = 0{,}316$. Скан по величине шага
даёт **внутренний оптимум** $s^\star \approx 0{,}6$ ($\langle\nu^{\mathrm{op}}\rangle = 0{,}290$), уверенно бьющий
замёрзший Роббинс–Монро (0,820) — «забывать в темпе дрейфа». Границы: $\eta \in [0, 0{,}008]$,
$\nu^{\mathrm{op}} \in [0, 1]$, ноль нарушений $I_{\mathrm{pred}} > I_{\mathrm{opt}}$ на 50000 точках (см.
`results_summary.txt`). Адиабатическая асимптотика S8.1 в этой точке ($\sigma^2/\lambda=10$) не тестируется —
её проверка в `../markov_drift_ou_iinf_adiab/`.

## Связь с `../markov_drift/`

PSP-симуляция в `../markov_drift/` — это *качественная* численная иллюстрация
§ 6.1; она не порождает BNT-подобный член $(K/2)\ln(\lambda t)$, потому что
пуассоновские переключения режима без повторного посещения латентных состояний
разрушают концентрацию постериора при каждом переключении (см. оговорку § 5.2 и
количественное обсуждение § 6.1). OU-симуляция здесь — *строгий* аналог: дрейф
непрерывен, параметр — один и тот же $\theta(t)$ на всём протяжении, и идеальный
байесовский обучающийся видит повторно посещаемую латентную переменную, то есть
структурные предпосылки S8.1 здесь выполнены. Однако при $\sigma = 0{,}1$ система
находится вне режима OU-concentration, поэтому BNT-логарифм $(K/2)\ln(\lambda t)$ в
этой параметризации не проявляется (см. раздел «Результат»); проверка самой
асимптотики S8.1 — в `../markov_drift_ou_iinf_adiab/`.
