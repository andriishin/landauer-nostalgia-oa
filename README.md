# Informational Nostalgia — open materials

*([Русская версия](./README.ru.md))*

Open-access materials for the article **«Informational Nostalgia: Memory Growth, the Irreducible Prediction Shortfall, and the Thermodynamics of Non-Stationary Learning»** (Russian primary; English translation included).

This is the companion to *«Vitality as the Efficiency of Self-Paid Self-Modeling»* ([andriishin/landauer-self-modeling-oa](https://github.com/andriishin/landauer-self-modeling-oa)), extending the predictive efficiency of self-modeling

```
η_v = I_pred / I_mem = 1 − ν
```

from the stationary regime to a **non-stationary** environment with growing memory. A system holds its model against an environment that drifts faster than the model is refreshed, so a growing fraction of the stored bits no longer predicts anything — *informational nostalgia*, measured operationally by the **prediction shortfall** ν^op (the fraction of the environment's predictable future the system misses). The framework applies under a *self-payment* condition — every held bit is paid for by the system's own free energy, its thermodynamic price set by a separate Still bound. Two results are proved: **Lemma 1** (holding a bit against thermodynamic erosion costs strictly positive power, with an explicit lower bound) and **Lemma 2** (a conditional result: under slow Ornstein–Uhlenbeck drift, any finite refresh rate, and a numerically verified additivity assumption, the prediction shortfall asymptotically stays above an explicit positive constant). Consequences: predictive capacity collapses above a critical nostalgia, and an optimal rate of complete memory reset exists.

## Recompiling the PDFs

Precompiled PDFs are included for direct reading. To rebuild from the `.tex` you need a TeX distribution with `pdflatex` and `bibtex` (TeX Live or MiKTeX). The Springer Nature template files (`sn-jnl.cls`, `sn-*.bst`) are bundled, so no further downloads are required. The reference style is `sn-mathphys-ay` (author–year). Standard cycle for any document, e.g. `main`:

```bash
cd paper
pdflatex main.tex
bibtex   main
pdflatex main.tex
pdflatex main.tex
```

(replace `main` with `supplementary`, `main.ru`, or `supplementary.ru` for the other documents). The figures in `paper/figs/` are required for compilation and are regenerated by the simulations.

## Reproducing the simulations

Each `simulations/<experiment>/` is self-contained:

```bash
cd simulations/markov_drift_ou && pip install -r requirements.txt && python main.py
```

Seeded with `SEED=20260524` and per-run offsets; on the same NumPy version each run reproduces the figures and the numbers in `results_summary.{txt,json}`. Each folder has a `README.md` (English) and `README.ru.md` (Russian) documenting the model, parameters, and the result it establishes.
